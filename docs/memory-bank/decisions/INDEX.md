# Decisions (ADRs)

Index des decisions d'architecture du projet.

## Format ADR

Chaque decision suit ce format :

```markdown
# [Titre de la decision]

**Date:** YYYY-MM-DD
**Status:** proposed | accepted | deprecated | superseded

## Contexte
[Pourquoi cette decision est necessaire]

## Decision
[Ce qu'on a decide]

## Consequences
- Positif : [avantages]
- Negatif : [inconvenients]
```
