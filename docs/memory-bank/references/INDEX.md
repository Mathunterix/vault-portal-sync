# References

Guides et apprentissages issus des sessions de travail.

## Qu'est-ce qu'une reference ?

Une reference est un guide distille depuis :
- Une session de travail complexe
- Une deep-search pertinente
- Un apprentissage technique important

Contrairement aux features (documentation de ce qui a ete fait), les references sont des **guides reutilisables** pour le futur.
